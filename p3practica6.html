Lista de viaje




La api que se implemento fue la de geolocalizacion, esta api nos pide acceso a la ubicacion para poder dar las coordenadas correspondientes a la ubicacion del dispositivo desde el cual se realiza la accion. La importancia de esta api en mi pagina, es que gracias a ella, el viajero,puede ver donde esta, en todo momento, para lograr no perderse y lograr hacer sus listas. Esta pagina tiene la utilidad de lograr organizar tus actividades, lugares por visitar y la lista de compras que deseas realizar, ademas de proporcionarte tu ubicacion, para que puedes actualizar y borrar datos de tus listas.
